# 🚀 Antigravity 遠端橋接器使用說明 (手機控制電腦版)

本工具可讓您人在外面時，透過手機網頁輸入指令，自動文字輸入到您電腦端的 Antigravity 對話框中。

## 🛠 啟動步驟
1.  在電腦端開啟終端機 (Terminal)。
2.  輸入以下指令啟動橋接伺服器：
    ```bash
    python3 /Users/tainanmac2021/.gemini/antigravity/playground/primal-sun/tools/remote_bridge/bridge.py
    ```
3.  畫面上會出現一個網址，例如：`http://192.168.1.5:8080`。
4.  拿起手機，在瀏覽器中輸入該動態網址。

## 📱 手機端操作
1.  進入網頁後，直接在文字框輸入您想叫 AI 執行的指令（例如：「重構 data.js，增加怪物等級」）。
2.  點擊 **「發送指令」**。
3.  您電腦上的 Antigravity 就會像被「自動打字」一樣收到指令並開始工作！

## ⚠️ 注意事項
- **視窗置頂**：發送時，請確保電腦上的 Antigravity 視窗沒有被最小化（腳本會自動嘗試將其置頂）。
- **權限需求**：第一次執行時，Mac 可能會要求「輔助功能 (Accessibility)」權限，請務必允許，腳本才能模擬鍵盤操作。
- **安全建議**：此服務僅供您的區域網路使用，請確保手機與電腦連在同一個 Wi-Fi (或使用您習慣的遠端穿透軟體)。
