# 怪物卡牌與介面重構完成

## ✅ 執行項目 No. [3103]

1.  **怪物資料結構 (Data Structure)**:
    *   **Desc**: 僅保留 **Flavor Text (背景描述)**。
    *   **Abilities_desc**: 新增欄位，專門記錄 **特殊能力描述** (如【進場】、【持續】效果)。

2.  **地城卡牌介面 (Dungeon Card UI)**:
    *   **排版**: 垂直堆疊
        1.  怪物名稱
        2.  ❤️ 生命值
        3.  🛡️ 護盾損耗
        4.  ✨ XP 獎勵
        5.  **描述區域**: 若有特殊能力則顯示特殊能力，否則顯示背景描述。

3.  **怪物詳情框 (Detail Modal)**:
    *   **上方**: 名稱、數值 (❤️/🛡️/✨)。
    *   **特殊能力**: (靠左對齊) 顯示 `abilities_desc`，若無則顯示「（無）」。
    *   **下方**: (靠左對齊) 顯示背景描述 `desc`。

請檢視地城中的怪物卡牌是否符合您的預期！
