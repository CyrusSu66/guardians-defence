# 卡牌資料與排版重構完成

## ✅ 執行項目 No. [2997]

1.  **資料結構調整**:
    *   **Desc**: 作為 **Flavor Text (背景描述)**，例如：「受過基礎訓練的民兵...」。
    *   **Abilities_desc**: 新增欄位，作為 **特殊能力描述**，例如：「⚔️ 戰鬥中：若裝備長矛...」。
    *   **Fallback**: 若無特殊能力，卡面會顯示 Desc。

2.  **市集與手牌排版更新**:
    *   **Row 1**: 卡牌名稱
    *   **Row 2**: 數值圖示 (💪1 🪄0 / ⚔️1 🪄0 ⚖️1) — 緊接在名稱下方。
    *   **Row 3**: 特殊能力描述 (若無則顯示背景描述)。
    *   **Bottom**: 費用 (💰) 或 價值 (🪙)。

3.  **詳情框 (Detail Modal) 更新**:
    *   **名稱**
    *   **背景描述**: (灰色斜體)
    *   **特殊能力**: (白色文字) — 顯示 `abilities_desc` 內容，若無則顯示「（無）」。
    *   **數值**: (原有排版)

請檢視新的排版是否符合您的設計需求！
